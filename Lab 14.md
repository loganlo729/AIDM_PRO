## Feature Description
In this version of the project, the user can create a character by entering character sheet information one field at a time. The program collects important character details such as name, class, race, level, and other related stats, then stores that information in `characters.json' so it can be used later in the project.

## How This Fits the Project
Character creation provides the base player information needed for later features. In future development, the saved character data will be used together with narration generation, game state management, and other AI-supported gameplay features involving tools. There will also be support for backstories and character descriptions.
